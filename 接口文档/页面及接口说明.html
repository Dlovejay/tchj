太仓海警支队页面和接口说明

  • 1 登录页 /page/

    ajax
    1. 用户登录接口 /login/
      post {username:用户名,password:密码,auto:[1,0]自动登录，1自动}
      return {code:...message:...}

  • 2 登出页 /page/

    本页面无任何数据提交，请在ctroller中清除session和cookie即可

  • 3 主框架页 /page/

    渲染数据
    1. 当前登录用户信息
      {username,tid,tname(用户类型名称),pid,pname(部门名称),jid,jname(职务名称)}

  • 4 部门管理页 /page/

    渲染数据
    1. 当前部门信息
      [{pid,pname,plevel},{...}...]
    ajax
    1. 新增部门接口 /login/
      post {pname:部门名称,plevel:部门级别[1,2] 其他数值判定为出错}
      return {code:...message:...,data:新增id}
    2. 修改部门接口 url
      post {pid,pname:部门名称}
      return {code:...message:...}
    3. 删除部门 url
      post {pid}
      return {code:...message:...}
